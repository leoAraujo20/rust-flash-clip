# 🦀 Rust FlashClip - CEULP/ULBRA

Bem-vindo ao material de apoio do curso introdutório de **Rust** para o evento **FlashClip**!

Este repositório foi criado pela turma de **Paradigmas de Linguagem de Programação** (1º Semestre de 2026) do **Centro Universitário Luterano de Palmas (CEULP/ULBRA)**. Aqui você encontrará exemplos práticos e desafios para aprender os fundamentos da linguagem.

## 📂 Organização do Curso

Os códigos estão divididos em módulos para facilitar o acompanhamento durante as demonstrações no compilador online.

### 🏁 [Módulo 1: Primeiros Passos](./modulo-1/)

- O básico para ver o Rust funcionando: "Hello World" e uma saudação ao evento.

### 🔢 [Módulo 2: Variáveis e Dados](./modulo-2/)

- Como o Rust lida com memória e dados:
  - Criação e exibição de variáveis.
  - Tipos de dados e conversões.
  - Mutabilidade (variáveis que podem ou não mudar).
  - Operações matemáticas e textos.
  - Entrada de dados pelo usuário.

### 🔄 [Módulo 3: Controle de Fluxo](./modulo-3/)

- Tomada de decisão e repetições:
  - `if`, `else if` e `else` (Estruturas de condição).
  - Loops com `for`, `loop` e `while`.

### 🏆 [Desafios Práticos](./desafios/)

Use estes arquivos como base (esqueletos) para resolver os problemas propostos:

- **Calculadora Personalizada:** Prática de operações e entradas.
- **Máquina de Luta:** Um desafio integrador com diversos conceitos aprendidos.
- **Jogo da Adivinhação:** Lógica de condições e repetições.
